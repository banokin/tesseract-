# OCR через Tesseract (Streamlit + FastAPI)

Приложение для распознавания текста с изображений с использованием OCR движка Tesseract.
Внутри Fastapi и интерфейс Streamlit

## Что делает проект

Вы загружаете изображение (PNG/JPG/JPEG).
- **FastAPI** принимает файл по эндпоинту `/ocr` и вызывает tesseract для извлечения текста.
- Результат распознавания отображается на странице в текстовом поле.

## Виртуальное окружение (`venv`)

Основная среда проекта — каталог **`venv`** в корне репозитория (уже указан в `.gitignore`).

```bash
# создать окружение
python3 -m venv venv

# активировать (macOS / Linux)
source venv/bin/activate

# Windows (PowerShell)
# .\venv\Scripts\Activate.ps1

pip install -r requirements.txt
```

Дальше все команды (`uvicorn`, `streamlit`, `pytest`) запускайте **с активированным `venv`**.

## Как запустить

1. Создайте и активируйте **`venv`** (см. выше), установите зависимости:  
   `pip install -r requirements.txt`

2. Запустите API:
   
   ```bash
   # разработка (один процесс, автоперезагрузка)
   uvicorn main:app --reload --host 0.0.0.0 --port 8000
   ```
   
   **Несколько пользователей одновременно:** тяжёлая работа (OCR, запись `.docx`) выполняется в **пуле потоков** и с **лимитом параллельных задач** (см. ниже). Для максимальной пропускной способности на многоядерном сервере запускайте **несколько процессов** uvicorn (без `--reload`):
   
   ```bash
   uvicorn main:app --host 0.0.0.0 --port 8000 --workers 4
   ```
   
   Переменные окружения (опционально):
   - `OCR_THREAD_POOL_WORKERS` — размер пула потоков на один процесс (по умолчанию: `min(32, CPU+4)`).
   - `OCR_MAX_CONCURRENT_JOBS` — сколько тяжёлых задач одновременно на процесс (по умолчанию: как у пула). Уменьшите, если не хватает RAM.

3. Запустите интерфейс в отдельном терминале:
 
   streamlit run streamlit_app.py
   
Страница откроется автоматически в браузере

После этого откройте страницу Streamlit и загрузите изображение через кнопку "Browse file"

Ниже изображения кнопка "Распознать текст"

Текст появится после нажатия кнопки "Распознать текст".


во время загрузки фотографии streamlit с помощью библиотеки request отправляет запрос на FastApi. FastApi с помощью tesseract  сканирует картинку на текст и возвращает текст на Streamlit

## Требования


- На системе должен быть установлен tesseract.
- Доступны языки русский и английский для распознания.
